# Module 5: Projects, Portfolio, And Job Strategy

## Why Projects Matter More Than Courses

Courses give you exposure.
Projects give you proof.

When switching into AI, employers and collaborators usually need evidence that you can:

1. frame a problem
2. choose an approach
3. build something real
4. evaluate it honestly
5. explain trade-offs

## What A Good AI Transition Portfolio Contains

### Project 1: Foundation Project

Build something simple but rigorous:

- classification or prediction project
- evaluation metrics and notebook
- short write-up on model quality and limits

### Project 2: Applied LLM Project

Build something useful:

- document assistant
- domain chatbot
- internal search / RAG app
- coding or workflow assistant

Must include:

1. prompt design
2. retrieval or context strategy
3. evaluation cases
4. limitations and failure analysis

### Project 3: Production-Oriented Project

Show engineering maturity:

- deployment
- monitoring or logs
- tests or eval harness
- user workflow or API design

## What To Put In Your README Or Case Study

For each project, explain:

1. the problem
2. why AI was the right tool
3. data or context inputs
4. model or system design
5. evaluation approach
6. failure modes
7. next improvements

## How To Reposition Yourself Professionally

### Resume Shift

Do not say only:

- "Interested in AI"

Say:

- built and evaluated AI-assisted workflow tools
- designed prompt and evaluation pipelines
- deployed model-backed applications
- implemented retrieval and grounding for domain-specific assistants

### Interview Strategy

Be ready to explain:

1. what you built
2. why you chose the approach
3. how you measured success
4. what failed and how you improved it
5. what trade-offs you made

## Hiring Reality

The strongest candidates are often not the ones with the most courses.

They are the ones who can combine:

1. practical build ability
2. clear thinking
3. evaluation discipline
4. role-relevant projects

That is the actual bridge from "learning AI" to "working in AI".
